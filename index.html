Hey!

I'm Alex Huard, a game Ddesigner, programmer, and game playing guy.

This website is to help compile and keep track of all of my projects I have worked on.
I have been making games as a hobby since I was in 5th grade using MMF2 to make games. As I've continued to make games for almost half my life, I have definitely changed styles several times and I have worked on countless ideas.

I primarily work in Gamemaker Studio but I also do occasional work in Unreal 4 and the Unity engine. Right now the main portion of this website is the Games section but I will eventually add a Tech Demo section where I can organize my little projects that don't become games.
